# Stock Marketplace

A basic entry-level stock marketplace featuring 2 stocks (AAPL & TSLA).

## Features

- Live price feed (prices fluctuate every 5 seconds)
- Buy and sell shares with a $10,000 starting balance
- Portfolio view with real-time gain/loss tracking
- Transaction history

## Stocks Available

| Symbol | Company     | Starting Price |
|--------|-------------|----------------|
| AAPL   | Apple Inc.  | $178.50        |
| TSLA   | Tesla Inc.  | $242.80        |

## Getting Started

```bash
npm install
npm start
```

Then open http://localhost:3000 in your browser.
